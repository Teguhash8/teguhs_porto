🌍 Visa Travels: Personal Web Design Project
📌 Tentang Proyek
Proyek ini adalah eksplorasi desain UI/UX dan branding personal saya untuk sebuah agensi perjalanan fiktif bernama Visa Travels. Fokus utama saya dalam proyek ini adalah menciptakan pengalaman visual yang mewah (luxury), minimalis, namun tetap fungsional bagi pengguna yang mencari perjalanan terkurasi.
