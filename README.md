# Teguhash | Creative Developer Portfolio

A luxurious, ultra-modern portfolio website built with Next.js 15, React 19, Tailwind CSS, and Framer Motion. This project is carefully designed and engineered from scratch to reflect minimalist aesthetics, high performance, and smooth animations.

## Features

- **Blazing Fast Performance**: Next.js App Router with Server Components and highly optimized assets.
- **Micro-Interactions**: Powered by Framer Motion, enabling beautiful scroll-triggered animations and fluid page transitions.
- **Modern Tech Stack**: Uses React 19, TypeScript, Tailwind CSS, and Lucide React icons.
- **Production Ready**: Zero linting errors, fully typed, tree-shaken, and optimized for Vercel deployment.
- **Responsive Layout**: Pixel-perfect on both mobile and desktop screens.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build for Production

To create an optimized production build, run:

```bash
npm run build
npm start
```

## Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/) & [GSAP](https://gsap.com/)
- **UI Components**: custom-built interfaces with modern UX paradigms
